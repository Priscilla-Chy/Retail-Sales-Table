# Retail Sales Table

## Objective
The objective of this project was to organise and analyse a retail sales dataset using basic Microsoft Excel features. The task focused on structuring raw data, applying filters, and calculating key summary metrics to support simple business insights.

## Dataset
The dataset contains retail sales information, including customer age, product details, sales values, and commission amounts. The data was provided in a raw format and required structuring before analysis.
